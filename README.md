# 📖 SCHOOL LIBRARY <a name="about-project"></a>

**School Library** is an app built with Ruby. It demonstrates Ruby's Object-Oriented Programming style (OOP)

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Ruby

### Key Features <a name="key-features"></a>

- Encapsultaion
- Inheriatance
- Polymophism
- Abstraction
- Can add new teacher and student
- Can create new book
- Save records of who borrowed a given book and when
